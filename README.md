Mario game project - Một game platformer kiểu Mario kinh điển được xây dựng bằng Python và Pygame.

mario-game/
├── main.py                    # File chính để chạy game
├── Core.py                    # Lớp lõi, vòng lặp game
├── Player.py                  # Lớp nhân vật chính Mario
├── Map.py                     # Quản lý map và level
├── MenuManager.py             # Quản lý menu
├── Sound.py                   # Quản lý âm thanh
├── Const.py                   # Hằng số toàn cục
├── Entity.py                  # Lớp cơ sở cho các entity
├── Camera.py                  # Hệ thống camera follow
├── Platform.py                # Nền tảng chơi game
├── Tube.py                    # Ống dẫn (tube/pipe)
├── Crate.py                   # Hộp/thùng chứa
├── Coin.py                    # Đồng tiền
├── Mushroom.py                # Power-up mushroom
├── Flower.py                  # Power-up fire flower
├── Goombas.py                 # Kẻ thù Goomba
├── Koopa.py                   # Kẻ thù Koopa
├── Fireball.py                # Quả fireball
├── Flag.py                    # Cờ hoàn thành level
├── GameUI.py                  # Giao diện người dùng
├── MainMenu.py                # Menu chính
├── LoadingMenu.py             # Menu loading
├── Text.py                    # Xử lý text
├── DebugTable.py              # Công cụ debug
├── Event.py                   # Hệ thống event
├── mario/                     # Thư mục chứa sprites và assets
│   └── Next/
│       └── images/            # Hình ảnh sprite Mario và kẻ thù
└── README.md                  # File hướng dẫn
